## Experiencias SQL:

Reportes de uso comercial relacionados a precios, ventas, maestro de artículos, maestro de proveedores, movimiento de stock y artículos, entre otros

Tareas SQL:

	- Relevamiento
	- Armado de consultas
	- Extracción, exploración y análisis de datos
	- Creación y generación automática de reportes

Evento: Correctivos y Evolutivos en los diferentes módulos y aplicaciones del ERP

Tareas SQL:

	- Inspección de Stored Procedures
	- Relevamientos de tablas y sus relacionales
	- Armados de SP's ejecutables para pruebas
	- Uso del Profiler y ejecución de Trace para la captura de datos en procesamiento
